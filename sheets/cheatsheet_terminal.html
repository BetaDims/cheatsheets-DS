# léeme primero

En toda la hoja, lo que va en cursiva con subrayado punteado es un marcador: no lo escribes tal cual, lo reemplazas por lo tuyo. Esto es lo que significa cada uno.

cmd · cmd1 · cmd2Un comando cualquiera. Los números indican el 1.º, 2.º… cuando se encadenan (cmd1 | cmd2).
file · archivoUn archivo concreto, p. ej. notas.txt.
carpetaUn directorio, p. ej. proyecto/.
origen · destinoLa ruta de la que copias/mueves y la ruta a la que va.
hostEl nombre o IP de una máquina, p. ej. google.com.
urlUna dirección web completa.
patronEl texto o la expresión que estás buscando.
PIDEl número que identifica un proceso (te lo da ps).
VAR · paqueteEl nombre de una variable de entorno · el nombre de un programa a instalar.
~ · . · ..Tu carpeta personal (home) · el directorio actual · el directorio padre.

¿Terminal, shell o CLI?

# tres cosas distintas
TerminalLa ventana donde escribes y lees texto. Es el «monitor»: por sí sola no ejecuta nada.
ShellEl programa que lee tu comando, lo ejecuta y devuelve el resultado. El «motor»: bash, zsh, fish.
CLILa línea de comandos: usar el equipo escribiendo órdenes en vez de hacer clic. Lo contrario de una GUI.

En una frase: la terminal es la ventana, el shell es quien obedece y la CLI es el idioma de texto con el que le hablas.

Atajos de teclado

Editar la línea sin ratón ni flechas. Los que de verdad ahorran tiempo.

TabAutocompleta comandos y rutas. Doble Tab = ver opciones.
↑ ↓Navega por el historial de comandos.
Ctrl + RBusca en el historial mientras escribes.
Ctrl + CCancela / interrumpe el proceso actual.
Ctrl + LLimpia la pantalla (como clear).
Ctrl + A Ctrl + ECursor al inicio / al final de la línea.
Alt + ← Alt + →Mueve palabra por palabra (en Mac, Option). = Alt+B/F.
Ctrl + UBorra toda la línea (todo lo de la izquierda del cursor).
Ctrl + KBorra desde el cursor hasta el final (lo de la derecha).
Ctrl + WBorra la palabra a la izquierda del cursor.
Alt + DBorra la palabra a la derecha del cursor.
Ctrl + YPega (yank) lo último que borraste con Ctrl+U/K/W.
Ctrl + TIntercambia los dos caracteres junto al cursor.
Ctrl + _Deshace la última edición — el verdadero «undo».
WSL + ZshSon los atajos del modo emacs (el de serie). Ctrl + _ se teclea como Ctrl + Shift + -. Si Alt no responde en Windows Terminal, usa Alt+B/F/D, que siempre funcionan. Ojo: Ctrl+Z no es undo — suspende procesos.

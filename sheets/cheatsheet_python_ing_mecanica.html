Arrays y vectorización

B

El array de NumPy reemplaza los bucles: operas sobre vectores completos de golpe (rápido y legible). Ideal para tablas de valores (tiempo, posición, esfuerzo).

ClaveLas operaciones son elemento a elemento. Usa np.sin, np.sqrt, etc., no las del módulo math (que no aceptan arrays).

Álgebra lineal

I

Resolver sistemas $A\vec x=\vec b$ (equilibrio de armaduras, redes) es una línea. Nunca inviertas la matriz a mano: usa np.linalg.solve.

Evitanp.linalg.inv(A) @ b es más lento y menos estable que solve.

Datos de ensayos (CSV)

I

Para datos experimentales (sensores, ensayos de tracción, registros de vibración), Pandas lee CSV/Excel, filtra, agrupa y resume con pocas líneas.

Útildf.describe() da media, desviación y percentiles al instante. df.groupby('material').mean() resume por categoría. Exporta con df.to_excel(...) para informes.

Integrar y derivar

I

Trabajo, área bajo una curva, centroides: integración numérica. quad para funciones, trapezoid para datos muestreados.

Raíces y optimización

I

Para ecuaciones no lineales (Colebrook, equilibrio de fases) usa fsolve; para minimizar (peso, costo) minimize.

EDOs (dinámica)

A

Vibración, caída con arrastre, sistemas masa-resorte-amortiguador: solve_ivp integra $\dot{\vec y}=f(t,\vec y)$. Ejemplo: oscilador amortiguado $m\ddot x+c\dot x+kx=0$.

Anatomía de un gráfico

B

El patrón fig, ax (orientado a objetos) es el recomendado: controlas ejes, rejilla, etiquetas y leyenda. Siempre rotula con unidades.

Gráfico de ingeniería — la firma

I

El resultado típico: una curva esfuerzo-deformación trazada con Matplotlib, con rejilla, ejes rotulados y los puntos clave anotados (límite elástico, fluencia, resistencia última, rotura). Es lo que entregas en un informe.

matplotlib · stress-strain curve
σ-ε de un acero dúctil (salida de plt)
Generado con ax.plot(eps, sigma) + ax.annotate(...). La pendiente elástica es el módulo $E$; el área bajo la curva es la tenacidad.
